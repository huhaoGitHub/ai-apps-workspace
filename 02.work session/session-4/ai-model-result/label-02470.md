# 学生管理系统测试任务方案

## 一、系统架构概览

### 后端技术栈
- **框架**: Spring Boot 3
- **ORM**: JPA (Hibernate)
- **数据库**: MySQL
- **认证**: JWT (Bearer Token)
- **端口**: 8080

### 前端技术栈
- **框架**: Vue 3
- **UI库**: Element Plus
- **HTTP客户端**: Axios
- **路由**: Vue Router

| 模块 | 接口路径 | 功能 |

### 任务1：登录鉴权测试

#### 1.1 登录功能测试
| 测试项 | 测试步骤 | 预期结果 |
|--------|----------|----------|
| 正常登录 | 输入正确用户名密码 | 返回200，包含token、username、realName、role |
| 错误用户名 | 输入不存在的用户名 | 返回错误："用户名或密码错误" |
| 错误密码 | 输入正确用户名+错误密码 | 返回错误："用户名或密码错误" |

| 禁用账号 | 使用禁用账号登录 | 返回错误："账号已禁用" |

#### 1.2 JWT Token 测试
| 测试项 | 测试步骤 | 预期结果 |
|--------|----------|----------|
| Token格式 | 检查返回的token | 应为JWT格式：header.payload.signature |
| Token有效期 | 解析token的exp字段 | 应为24小时（86400000ms） |
| Token包含信息 | 解析payload | 应包含sub(用户名)、role(角色) |

#### 1.3 Bearer Token 认证测试
| 测试项 | 测试步骤 | 预期结果 |
|--------|----------|----------|
| 正确Token | 请求头携带 `Bearer {token}` | 正常访问受保护接口 |
| 无Token | 不携带Authorization头 | 返回401，`{"code":401,"message":"未登录"}` |
| 错误Token格式 | 携带 `InvalidToken`（无Bearer前缀） | 返回401 |
| 过期Token | 使用过期的token | 返回401 |
| 篡改Token | 修改token内容 | 返回401 |

```javascript
// 1. 登录获取Token
describe('Auth API', () => {
  it('should login successfully with valid credentials', async () => {
    const response = await request.post('/api/auth/login')
      .send({ username: 'admin', password: 'admin123' });
    
    expect(response.status).toBe(200);
    expect(response.body.code).toBe(0);
    expect(response.body.data.token).toBeDefined();
    // 保存token供后续测试使用
    global.token = response.body.data.token;
  });
});

// 2. 学生CRUD测试
describe('Student CRUD', () => {
  let studentId;
  
  it('should create a student', async () => {
    const response = await request.post('/api/students')
      .set('Authorization', `Bearer ${global.token}`)
      .send({
        studentNo: '2024001',
        name: '测试学生',
        gender: '男',
        clazz: { id: 1 }
      });
    
    expect(response.status).toBe(200);
  });
  
  it('should get student list with pagination', async () => {
    const response = await request.get('/api/students?page=1&size=10')
      .set('Authorization', `Bearer ${global.token}`);
    
    expect(response.body.data.records).toBeDefined();
    expect(response.body.data.total).toBeDefined();
  });
});

// 3. 401场景测试
describe('Unauthorized Access', () => {
  const protectedEndpoints = [
    { method: 'GET', url: '/api/students' },
    { method: 'GET', url: '/api/clazz' },
    { method: 'GET', url: '/api/courses' },
    { method: 'GET', url: '/api/scores' },
  ];
  
  protectedEndpoints.forEach(endpoint => {
    it(`should return 401 for ${endpoint.method} ${endpoint.url} without token`, async () => {
      const response = await request[endpoint.method.toLowerCase()](endpoint.url);
      expect(response.status).toBe(401);
      expect(response.body.code).toBe(401);
    });
  });
});

// 4. 成绩录入测试
describe('Score Management', () => {
  it('should reject duplicate score entry', async () => {
    // 先录入一次
    await request.post('/api/scores')
      .set('Authorization', `Bearer ${global.token}`)
      .send({ studentId: 1, courseId: 1, score: 85, semester: '2024-2025-1' });
    
    // 再次录入同一学生同一课程
    const response = await request.post('/api/scores')
      .set('Authorization', `Bearer ${global.token}`)
      .send({ studentId: 1, courseId: 1, score: 90, semester: '2024-2025-1' });
    
    expect(response.body.message).toContain('已有成绩记录');
  });
  
  it('should reject score out of range', async () => {
    const testCases = [-1, 101, null];
    for (const score of testCases) {
      const response = await request.post('/api/scores')
        .set('Authorization', `Bearer ${global.token}`)
        .send({ studentId: 1, courseId: 2, score, semester: '2024-2025-1' });
      expect(response.body.message).toContain('0-100');
    }
  });
});
```
